Resources and Development
=========================

Bug Tracker
-----------

Superguard has a bug tracker where you may report any bugs or other
errors you find.  Please report bugs to the `GitHub issues page
<https://github.com/daqianai/superguard/issues>`_.

Version Control Repository
--------------------------

You can also view the `Superguard version control repository
<https://github.com/daqianai/superguard>`_.

Contributing
------------

`Pull requests <https://help.github.com/articles/about-pull-requests/>`_
can be submitted to the Superguard repository on GitHub.
